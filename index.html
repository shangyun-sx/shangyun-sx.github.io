启程：我的博客诞生记

「 你好，世界 」

终于，在无数次“想要一个自己的博客”的念头闪过之后，今天，我按下了那个关键的 hexo deploy 命令。

看着命令行窗口里飞速滚动的代码，我知道，这片属于我的小小天地，正式在互联网的世界里 「启程」了。

🛠️ 为什么搭建这个博客？

我搭建这个博客，主要出于三个目的：

  1. 记录与分享：记录下学习、工作和生活中的思考与收获。如果能帮到偶然路过这里的你，那将是我的荣幸。
  2. 沉淀知识：把碎片化的信息整理成体系化的笔记。好记性不如烂笔头，Ctrl + S 是最好的记忆延伸。
  3. 练习表达：用写作倒逼自己更好地理解和消化知识。“教”是最好的“学”。

🌐 技术选型：为什么是 Hexo？

在众多博客框架中，我最终选择了 Hexo，原因很简单：

  • 极速生成：基于 Node.js，生成静态页面的速度快得惊人。
  • 简单易用：一条命令就能创建文章、启动服务器、部署上线。
  • 高度自由：有海量的主题和插件，可以随心所欲地定制。
  • 完全免费：借助 GitHub Pages 服务，可以零成本地托管整个博客。
